Using CSS Custom Properties for theming in modern web applications

Girish Gopaul

Mar 14, 2023 • 1 min read

CSS Custom Properties (aka CSS Variables) have been around for quite some time now. Though adoption has been slow (as for most w3c standards ðŸ˜…), we can now see it in action in most web apps (YouTube, Facebook, etc...).

But what are CSS Custom Properties exactly?

As the name states, they are custom user defined properties that can be defined on pages and used in any stylesheet on the page (irrespective of order of definition).

Once defined on a page, we can make use of these custom properties on any stylesheet on that page.

button.primary {
  background-color: var(--primary-color);
}

Interesting... But why use custom properties instead of our favorite CSS preprocessor like sass ?

  • Firstly because it does not require any kind of pre-processing or compiling.
  • More importantly because it can be used in any stylesheet and not limited to the context of the preprocessor.
